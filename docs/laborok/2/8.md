53. Pipeline architektúra – az osztály felépítése:

    ```
    StreamingPipeline
    ├── _produce_events(n)         → N db OrderEvent → "orders-pipeline" topic
    ├── _consume_and_validate()    → JSON Schema validáció, érvénytelen → DLQ
    ├── _aggregate_windows()       → TumblingWindowAggregator (10 s ablakok)
    ├── _export_parquet()          → aggregált ablakok → Parquet fájl
    └── start(n)                   → sorban: produce → consume → aggregate → export
    ```

    !!! note "Osztálymetódusok konvenciója"
        Az aláhúzással kezdődő metódusok (`_produce_events`, `_consume_and_validate`, stb.) Python konvenció szerint „privát" metódusok – ezeket csak az osztályon belülről szabad hívni (a `start()` metódusból). Az osztály felhasználója csak a `start()` metódust hívja.

        ```python
        def _consume_and_validate(self) -> list:
            consumer = KafkaConsumer(
                PIPELINE_TOPIC,
                bootstrap_servers=BOOTSTRAP,
                group_id="pipeline-validator",
                auto_offset_reset="earliest",
                enable_auto_commit=True,
                value_deserializer=safe_deserialize,
                consumer_timeout_ms=8000,
            )
            dlq_producer = KafkaProducer(
                bootstrap_servers=BOOTSTRAP,
                value_serializer=lambda v: json.dumps(v).encode("utf-8"),
            )
            valid_msgs = []
            valid = invalid = 0
            for msg in consumer:
                if isinstance(msg.value, dict):
                    try:
                        jsonschema.validate(instance=msg.value, schema=ORDER_SCHEMA)
                        valid_msgs.append(msg.value)
                        valid += 1
                    except jsonschema.ValidationError as e:
                        dlq_producer.send(DLQ_TOPIC, value={
                            "original": msg.value,
                            "error": e.message,
                            "ts": datetime.utcnow().isoformat(),
                        })
                        invalid += 1
            dlq_producer.flush()
            dlq_producer.close()
            consumer.close()
            print(f"Validálva: {valid} OK, {invalid} DLQ")
            return valid_msgs
        ```
