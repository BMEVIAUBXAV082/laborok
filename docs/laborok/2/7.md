# 7. feladat – Parquet export streamből

A streaming rendszerek kimenetét gyakran Parquet fájlokban tároljuk hosszú távon (data lake, HDFS, S3). A Parquet egy oszlopalapú bináris formátum, amely kiválóan tömörít, és OLAP lekérdezéseknél sokkal gyorsabb, mint a CSV. Ebben a feladatban egy **micro-batch** mintát implementálunk: N üzenetet összegyűjtünk memóriában, majd egy Parquet fájlba írjuk. Ezt `batch_id` alapján sorszámozzuk.

!!! note "Micro-batch minta"
    A micro-batch egy kompromisszum a valódi streaming és a klasszikus batch feldolgozás között. Ahelyett, hogy minden üzenetet azonnal fájlba írnánk (sok kis fájl → lassú olvasás), vagy végtelenül várnánk (nagy memória-igény), N-enként mentünk. Az N értéke (batch méret) hangolható: kisebb → kisebb latencia, nagyobb → jobb I/O hatékonyság.

    ```python
        bootstrap_servers=BOOTSTRAP,
        value_serializer=lambda v: json.dumps(v).encode("utf-8"),
    )
    # order_id: 400–499, hogy ne ütközzön az előző feladatokkal
    for i in range(400, 500):

            "event_time": datetime.utcnow().isoformat(),

        producer.send("orders", value=msg)
    producer.flush()
    producer.close()
    print("100 üzenet elküldve.")
    ```

        ```python
        def save_to_parquet(df: pd.DataFrame, batch_id: int) -> str:
            filename = f"{OUTPUT_DIR}/batch_{batch_id:03d}.parquet"
            df.to_parquet(filename, index=False)   # pandas → Parquet fájl (pyarrow háttérrel)
        ```

        **Mi a Parquet előnye a CSV-vel szemben?** Ugyanazok az adatok Parquet-ban általában 5–10× kisebbek, és az olvasás (különösen csak néhány oszlop esetén) 10–100× gyorsabb – mert az oszlopos tárolás lehetővé teszi, hogy csak a szükséges oszlopokat olvasson a lemezről.

48. Mentsük az összes üzenetet 3 batch-ben:

    ```python
    # Elosztjuk az összes rekordot 3 egyenlő részre
    batch_size = len(all_records) // 3   # Egész osztás → nincs töredék

    for batch_id in range(3):
        start = batch_id * batch_size
        # Az utolsó batch-nél a maradék is belekerül (len(all_records) helyett batch_size)
        end = start + batch_size if batch_id < 2 else len(all_records)
        batch_df = pd.DataFrame(all_records[start:end])  # Lista szelet → DataFrame
        save_to_parquet(batch_df, batch_id + 1)           # batch_id: 1, 2, 3 (nem 0, 1, 2)

    print("\nMind a 3 batch elmentve.")
    ```

    ```python
    # List comprehension: minden .parquet végű fájlt visszaolvasunk DataFrame-ként
    dfs = [pd.read_parquet(os.path.join(OUTPUT_DIR, f)) for f in files if f.endswith(".parquet")]

    # pd.concat: több DataFrame-et fűz össze egymás után soronként

    combined = pd.concat(dfs, ignore_index=True)
    ```
