# 5. feladat – Stream windowing és aggregáció

A batch feldolgozástól eltérően a streaming rendszerekben az adatok folyamatosan érkeznek, és nem lehet megvárni, amíg „mindenki megérkezik". A **windowing** (ablakozás) technikával az eseményeket időablakokba osztjuk, és ablakszinten aggregálunk. Így kapjuk meg pl. az „elmúlt 10 másodperc összes rendelésének bevételét".

!!! note "Tumbling window"
    A tumbling (bukó) ablak **fix méretű, nem átfedő** időszeleteket jelent. Például 10 másodperces tumbling window esetén az ablakok: 0–10 s, 10–20 s, 20–30 s, stb. Minden esemény **pontosan egy** ablakba kerül, az `event_time` mezője alapján. Más típusok: sliding window (átfedő), session window (aktivitás-alapú).

33. Implementáljuk a `_window_key` és `add` metódusokat:

    ```python

    #   1. event_time_iso string → datetime objektum:
    #      dt = datetime.fromisoformat(event_time_iso)
    #      (A fromisoformat() megérti a "+00:00" timezone jelzőt is)

    #   2. datetime → Unix timestamp (másodpercek 1970-01-01 óta):
    #      ts = dt.timestamp()
    #      (pl. 1710411612.345)

    #   3. Lefelé kerekítés a legközelebbi ablakhatárra:

    #      Példa: ts=1710411617, window_size=10 → floor(1710411617/10)*10 = 1710411610
    #      Ez azt jelenti: az ablak a 1710411610-es timestamptól indul (UTC időben)
    #
    # TODO: Az add metódus lépései:
    #   1. key = self._window_key(msg["event_time"])
    #   2. Ha key még nincs a self.windows dictben:

    #   3. self.windows[key]["count"] += 1

    #      (Egy üzenet bevétele = darabszám × egységár)
    ```

    ??? tip "Megoldási útmutató – kattints a tippekért"
        ```python
        def _window_key(self, event_time_iso: str) -> int:
            dt = datetime.fromisoformat(event_time_iso)   # str → datetime (timezone-aware is működik)
            ts = dt.timestamp()                            # datetime → float Unix timestamp
            return math.floor(ts / self.window_size) * self.window_size  # Ablak kezdetére kerekít

        def add(self, msg: dict) -> None:
            key = self._window_key(msg["event_time"])      # Melyik ablakba esik ez az üzenet?
            if key not in self.windows:
                self.windows[key] = {"count": 0, "total_revenue": 0.0}  # Új ablak inicializálása
            self.windows[key]["count"] += 1
            self.windows[key]["total_revenue"] += msg["quantity"] * msg["unit_price"]
        ```

        **Miért egész szám az ablakkulcs?** Az egész Unix timestamp jól kereshető és összehasonlítható dictben. A `math.floor()` garantálja, hogy az ablak határán lévő esemény mindig ugyanabba az ablakba kerül.

    ```python
    #             # Hány üzenet esett ebbe az ablakba
    #         "total_revenue": round(value["total_revenue"], 2),
    #             # Összes bevétel, 2 tizedesre kerekítve
    #         "avg_revenue": round(value["total_revenue"] / value["count"], 2),
    #             # Átlagos üzenetenkénti bevétel
    #     })
    #
    #   Rendezzük a listát window_start szerint: result.sort(key=lambda x: x["window_start"])
    #   Adjuk vissza: return result
    ```

    ??? tip "Megoldási útmutató – kattints a tippekért"
        ```python
        def flush(self) -> list:
            result = []
            for key, value in self.windows.items():
                result.append({
                    "window_start": datetime.fromtimestamp(key, tz=timezone.utc).isoformat(),
                    "count":         value["count"],
                    "total_revenue": round(value["total_revenue"], 2),
                    "avg_revenue":   round(value["total_revenue"] / value["count"], 2),
                })
            result.sort(key=lambda x: x["window_start"])   # Időrendi sorrend
            return result
        ```

        ```python
        consumer = KafkaConsumer(
            "orders-windowed",
            bootstrap_servers=BOOTSTRAP,
            auto_offset_reset="earliest",
            value_deserializer=lambda v: json.loads(v.decode("utf-8")),
            consumer_timeout_ms=5000,
        )

        aggregator = TumblingWindowAggregator(window_size=WINDOW_SIZE_SEC)
        n = 0
        for msg in consumer:
            if isinstance(msg.value, dict) and "event_time" in msg.value:
                aggregator.add(msg.value)
                n += 1

        window_results = aggregator.flush()
        consumer.close()
        print(f"Feldolgozott üzenetek: {n}")
        ```
