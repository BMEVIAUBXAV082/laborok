### 10.2 Boto3 vs Terraform összehasonlítás

Töltsd ki az alábbi táblázatot (Markdown cellában a notebookban, vagy szövegfájlban):

| Szempont | boto3 | Terraform |
|---------|-------|-----------|
| Vezérlési stílus | Imperatív (hogyan) | Deklaratív (mit) |
| Idempotencia | Manuálisan kell kezelni | Beépített (plan/apply) |
| State kezelés | Nincs state | terraform.tfstate |
| Legjobb felhasználás | Egyszeri műveletek, pipeline-triggerek | Infrastruktúra életciklusa |
| Learner Lab | Credentials azonnal beállítható | Terraform init után azonnal megy |

import boto3
