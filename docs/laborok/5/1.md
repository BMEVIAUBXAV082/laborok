### 1.1 Credentials ellenőrzése

```python
import boto3

sts = boto3.client('sts', region_name='us-east-1')
identity = sts.get_caller_identity()

print(f"Account ID : {identity['Account']}")
print(f"User ARN   : {identity['Arn']}")
print(f"User ID    : {identity['UserId']}")

# Ellenőrzés: LabRole-lal kell bejelentkezve lenni
assert 'LabRole' in identity['Arn'], "HIBA: Nem LabRole! Ellenőrizd a credentials fájlt."
print("\n✓ LabRole azonosítva, mehetünk tovább!")
```

### 1.2 Bucket nevek definiálása

```python
# TODO: írd be a saját Neptun-kódodat (kisbetűkkel)!
NEPTUN = "______"   # pl. "abc123"

PREFIX = f"bme-de-{NEPTUN.lower()}"
REGION = "us-east-1"

BUCKETS = {
    "raw":           f"{PREFIX}-smart-city-raw",
    "processed":     f"{PREFIX}-smart-city-processed",
    "streaming":     f"{PREFIX}-smart-city-streaming",
    "query_results": f"{PREFIX}-smart-city-query-results",
}

print("Létrehozandó bucket-ek:")
for zone, name in BUCKETS.items():
    print(f"  [{zone:12s}] {name}")
```

### 1.3 S3 bucket-ek létrehozása boto3-mal

```python
s3 = boto3.client('s3', region_name=REGION)

for zone, bucket_name in BUCKETS.items():
    try:
        # us-east-1: LocationConstraint NÉLKÜL kell létrehozni!
        s3.create_bucket(Bucket=bucket_name)     # TODO: ellenőrizd, hogy a bucket_name változót használod
        print(f"✓ Létrehozva  : {bucket_name}")
    except s3.exceptions.BucketAlreadyOwnedByYou:
        print(f"  Már létezik : {bucket_name}")
    except Exception as e:
        print(f"✗ HIBA [{bucket_name}]: {e}")
```

!!! note "us-east-1 sajátossága"
    Más régiókban (`eu-west-1` stb.) a `create_bucket()` híváshoz szükséges a `CreateBucketConfiguration={'LocationConstraint': 'eu-west-1'}` paraméter. Az `us-east-1` az egyetlen kivétel — itt TILOS megadni.

### 1.4 Lifecycle policy – raw bucket → Glacier 30 nap után

```python
s3.put_bucket_lifecycle_configuration(
    Bucket=BUCKETS["raw"],
    LifecycleConfiguration={
        "Rules": [{
            "ID":     "raw-sensors-to-glacier",
            "Status": "Enabled",
            "Filter": {"Prefix": "sensors/"},
            "Transitions": [{
                "Days":         None,   # TODO: írd be: 30

        }]

```

### 1.5 Ellenőrzés

```python
response = s3.list_buckets()
my_buckets = [b['Name'] for b in response['Buckets'] if NEPTUN in b['Name']]
print(f"A te bucket-jeid ({len(my_buckets)} db):")
for b in my_buckets:
    print(f"  {b}")

assert len(my_buckets) == 4, f"HIBA: {len(my_buckets)} bucket van, 4 kellene!"
print("\n✓ Mind a 4 bucket létezik!")
```
